code for Full-range Physical Adversarial Texture Against Person Detector

1. Requirements

All the codes are tested in the environment as therequirement.txt

2. Preparation

You need to download the necessary resources and put them in the right locations
2.1 Download INRIAPerson and put it in .\data\INRIAPerson
2.2 Download weights of yolo and put it in .\data\models and .\pytorchyolo\weights
2.3 Download cfg of yolo and put it in .\data\models

3. Run RFA_tex_training.py 

4. The well-trained RFA-Tex texture generation model is in .\pretrained
